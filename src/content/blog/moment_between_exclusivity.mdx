## moment between exclusivity

### Solution

```javascript
// proposal 1
m.isBetween(a, b, '()') // both excluded
m.isBetween(a, b, '[)') // start included, end excluded

// proposal 2
m.isBetween(a, b, '+') // both included
m.isBetween(a, b, '+-') // start included, end excluded
```
