## python average of two lists by row

### Solution

```python
# Basic syntax:
[(a + b)/2 for a, b in zip(list_1, list_2)]

# Example usage:
list_1 = [1, 2, 3]
list_2 = [4, 5, 6]
[(a + b)/2 for a, b in zip(list_1, list_2)]
--> [2.5, 3.5, 4.5]
```
