## how to Take Matrix input from user in Python

### Solution

```python
matrix = []
n = int(input("Enter the number of row = "))
for i in range(n):
    row = []
    print("Enter the all values of -", (i+1), "no. row with enter", end='')
    for j in range(n):
        row.append(int(input()))
    matrix.append(row)
print(matrix)
```
