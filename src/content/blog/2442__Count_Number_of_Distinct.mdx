## 2442. Count Number of Distinct Integers After Reverse OperationsMy SubmissionsBack to Contest

### Solution

```python
class Solution:
    def countDistinctIntegers(self, nums: List[int]) -> int:
        reverseList = []
        for i in nums:
            temp = str(i)
            temp = temp[::-1]
            reverseList.append(int(temp))
        nums = nums + reverseList
        return len(set(nums))
```
