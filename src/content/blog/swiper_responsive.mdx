## swiper responsive

### Solution

```javascript
breakpoints: {
    // when window width is >= 320px
    320: {
      slidesPerView: 2,
      spaceBetween: 20
    },
    // when window width is >= 480px
    480: {
      slidesPerView: 3,
      spaceBetween: 30
    },
  }
```

**Try other methods by searching on the site. That is if this doesn't work**
