## html boilerplate

### Solution

```html
<!doctype html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<link rel="stylesheet" href="Index.css" />
		<title>Title</title>
	</head>
	<body>
		<script src="Index.js"></script>
	</body>
</html>
```
