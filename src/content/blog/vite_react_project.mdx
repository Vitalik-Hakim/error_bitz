## vite react project

### Solution

```shell
# Create a React Project with Vite:
npm create vite@latest my-react-app -- --template react
cd my-react-app
npm install
npm run dev

# Create a React + TypeScript Project with Vite:
npm create vite@latest my-react-app-ts -- --template react-ts
cd my-react-app-ts
npm install
npm run dev
```
