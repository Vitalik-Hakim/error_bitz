## ubuntu error: EACCES: permission denied, symlink `../lib/node_modules/yarn/bin/yarn.js` -> `/usr/local/bin/yarn`

### Solution

```shell
WORKING SOLUTION

sudo chown -R $USER /usr/local

This will fix the issue.
```
