## c# get vector2 distance

### Solution

```csharp
//For 2D Unity C#
float GetDistance(Vector2 posA, Vector2 posB) {
	float dx = posA.x - posB.x;
    float dy = posA.y - posB.y;
    float distance = Mathf.Sqrt(dx * dx + dy * dy);
    return distance;
}
```
