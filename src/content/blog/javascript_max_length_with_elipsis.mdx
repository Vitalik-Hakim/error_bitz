## javascript max length with elipsis

### Solution

```javascript
function truncate_with_ellipsis(s, maxLength) {
	if (s.length > maxLength) {
		return s.substring(0, maxLength) + '...'
	}
	return s
}
alert(truncate_with_ellipsis('hello how are you today?', 9)) //hello how...
```
