## linux tar gz

### Solution

```shell
# Basic syntax:
# Compress:
tar -czvf <output_file> <directory_to_zip>

# Uncompress / extract:
tar -zxvf <tar_gz_file>

# View contents without extracting:
tar -tf <tar_gz_file>
```
