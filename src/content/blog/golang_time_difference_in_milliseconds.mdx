## golang time difference in milliseconds

### Solution

```go
package main
import (
        "log"
        "time"
)
func main() {
        start := time.Now().UnixNano() / int64(time.Millisecond)
        // do something
        time.Sleep(1 * time.Second)
        end := time.Now().UnixNano() / int64(time.Millisecond)
        diff := end - start
        log.Printf("Duration(ms): %d", diff)
}
```
