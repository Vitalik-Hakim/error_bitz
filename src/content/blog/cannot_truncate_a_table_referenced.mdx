**The solution for this is noted below**

## cannot truncate a table referenced in a foreign key constraint

### Solution

```sql
SET FOREIGN_KEY_CHECKS = 0;
TRUNCATE table $table_name;
SET FOREIGN_KEY_CHECKS = 1;
```
