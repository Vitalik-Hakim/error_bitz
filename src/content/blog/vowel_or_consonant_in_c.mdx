## vowel or consonant in c

### Solution

```c
// vowel or consonant check program
#include <stdio.h>
int main()
{
    char ch;
    ch = getchar();
    if (ch == 'a' || ch == 'e' || ch == 'i' || ch == 'o' || ch == 'u')
    {
        printf("%c is Vowel\n", ch);
    }
    else
    {
        printf("%c is consonant\n", ch);
    }
    return 0;
}
```
