## how to check if an element starts with class in jquery

### Solution

```typescript
/*SM*/

//Try using :
$('[class^=info_]') //Classes Starts with  info_
$('[class*=info_]') // Classes which contains info_

//instead of hasClass
if ($('label').is('[class*=info_]')) {
} //Contains
//or
if ($('label').is('[class^=info_]')) {
} //Starts with
```
