## php.ini settings

### Solution

```php
/usr/local/etc/php/7.3/php.ini
  
max_execution_time = 3600
max_input_time = 3600
max_input_vars = 10000
memory_limit = 1024M
post_max_size = 1024M
upload_max_filesize = 1024M
max_file_uploads = 200
short_open_tag = On
```
**Try other methods by searching on the site. That is if this doesn't work**
