## javascript falsy check

### Solution

```javascript
//Check if a value is falsy

// Short version
const isFalsy = (value) => !value

// Long version
const isFalsy = (value) => {
	if (
		value === null ||
		value === undefined ||
		value === 0 ||
		value === false ||
		value === NaN ||
		value === ''
	) {
		return true
	}
	return false
}
```
