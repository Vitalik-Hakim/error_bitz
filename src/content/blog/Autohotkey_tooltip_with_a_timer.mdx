## Autohotkey tooltip with a timer to remove itself

### Solution

```whatever
TimedToolTip("Example text",3000) ;In miliseconds

;-----Functions-----;
TimedToolTip(text,time)
{
	ToolTip, %text%
	SetTimer, RemoveToolTip, %time%
	return
}

;---Subroutines---;
RemoveToolTip:
	ToolTip
	return
```
