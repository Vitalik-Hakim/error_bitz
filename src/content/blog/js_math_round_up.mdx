## js math round up

### Solution

```javascript
console.log(Math.ceil(0.2))
// expected output: 1

console.log(Math.ceil(4))
// expected output: 4

console.log(Math.ceil(7.004))
// expected output: 8

console.log(Math.ceil(-7.004))
// expected output: -7
```
