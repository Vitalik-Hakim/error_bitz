## rotation turtle python

### Solution

```python
import turtle
t = turtle.Turtle()

#Setting Rotation
t.setheading(0) # This Faces Right
t.setheading(180) # This Faces Left
t.setheading(90) # This Faces Up
t.setheading(270) # This Faces Down

###################################
# Values are measured in degrees #
###################################
```
