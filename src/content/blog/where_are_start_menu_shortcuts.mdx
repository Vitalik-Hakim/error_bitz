## where are start menu shortcuts stored

### Solution

```whatever
On Windows
Start Menu items are mainly stored in the following directories

->  %AppData%\Microsoft\Windows\Start Menu (User)
->  %ProgramData%\Microsoft\Windows\Start Menu (Global)
```
