## find particular column or table name from whole db

### Solution

```sql
SELECT COLUMN_NAME
FROM INFORMATION_SCHEMA.COLUMNS
WHERE TABLE_NAME like '%ClaimProfessional%'
AND COLUMN_NAME LIKE '%payer%';
```
