## install mongodb compass on fedora

### Solution

```shell
#Download MongoDB Compass

wget https://downloads.mongodb.com/compass/mongodb-compass-1.33.0.x86_64.rpm

#Install MongoDB Compass

sudo yum install mongodb-compass-1.33.0.x86_64.rpm

#Start MongoDB Compass

mongodb-compass
```
