## logout in laravel

### Solution

```php
//route
Route::get('logout',[AuthController::class, 'logout'])->name('logout');

//logout function
use Auth,Session;
public function logout(){
  Auth::logout();
  Session::flush();
  return redirect()->route('login');
}
```
