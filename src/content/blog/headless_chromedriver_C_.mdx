## headless chromedriver C#

### Solution

```csharp
ChromeOptions options = new ChromeOptions();

options.AddArguments(new List<string>()
{
  "--silent-launch",
  "--no-startup-window",
  "no-sandbox",
  "headless"
});

ChromeDriver driver = new ChromeDriver(options);
```
