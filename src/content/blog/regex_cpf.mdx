## regex cpf

### Solution

```javascript
// CPF
/^\d{3}\.\d{3}\.\d{3}-\d{2}$/

// Para CNPJ
/^\d{2}\.\d{3}\.\d{3}\/\d{4}\-\d{2}$/

// Para ambos ao mesmo tempo
/(^\d{3}\.\d{3}\.\d{3}\-\d{2}$)|(^\d{2}\.\d{3}\.\d{3}\/\d{4}\-\d{2}$)/
```
