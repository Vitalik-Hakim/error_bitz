## jquery if element is clicked

### Solution

```javascript
$('#element').click(function () {
	var $this = $(this)
	if ($this.data('clicked')) {
		func(some, other, parameters)
	} else {
		$this.data('clicked', true)
		func(some, parameter)
	}
})
```
