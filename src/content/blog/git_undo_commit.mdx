**The solution for this is noted below**

## git undo commit

### Solution

```shell
# Uncommit the changes
git reset --soft HEAD~1

# Completely delete the changes
git reset --hard HEAD~1
```
**Try other methods by searching on the site. That is if this doesn't work**
