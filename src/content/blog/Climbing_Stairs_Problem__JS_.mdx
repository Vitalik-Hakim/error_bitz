## Climbing Stairs Problem (JS)

### Solution

```javascript
var climbStairs = function (n) {
	if (n == 1 || n == 0) return 1 // our base cases

	let first = 1
	let second = 2

	for (let i = 3; i <= n; i++) {
		let third = first + second
		first = second
		second = third
	}
	return second
}
```
