## get current epoch time rust

### Solution

```whatever
use std::time::{SystemTime, UNIX_EPOCH};

fn main() {
    let start = SystemTime::now();
    let since_the_epoch = start
        .duration_since(UNIX_EPOCH)
        .unwrap();
    println!("{:?}", since_the_epoch);
}
```
