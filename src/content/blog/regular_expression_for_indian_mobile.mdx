## regular expression for indian mobile number

### Solution

```javascript
Use the following regex

^(\+91[\-\s]?)?[0]?(91)?[789]\d{9}$
This will support the following formats:

8880344456
+918880344456
+91 8880344456
+91-8880344456
08880344456
918880344456
```
