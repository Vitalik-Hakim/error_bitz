## debounce on search vue

### Solution

```javascript
debounce(func, delay) {
      let timeoutId
      return (...args) => {
        clearTimeout(timeoutId)
        timeoutId = setTimeout(() => {
          func.apply(this, args)
        }, delay)
      }
```
