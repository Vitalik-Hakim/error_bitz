## django 3.2 show messages template

### Solution

```python
<div class="container">
    {% if messages %}
            {% for message in messages %}
              <div class="alert alert-{{ message.tags }}">
                {{ message|safe }}
              </div>
            {% endfor %}
          {% endif %}
    {% block content %}{% endblock content %}
  </div>
```
