## select not null jq

### Solution

```shell
JSON='[
  {
    "id": 1,
    "name": "John"
  },
  {
    "id": 2,
    "name": null
  }
]'
echo $JSON | jq '.[] | select(.name != null)'

# Output
{
  "id": 1,
  "name": "John"
}

echo $JSON | jq '.[] | select(.name != null) | .name'

# Output
"John"
```
