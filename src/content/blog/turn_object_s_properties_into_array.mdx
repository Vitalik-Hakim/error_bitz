## turn object`s properties into array of objects js

### Solution

```javascript
let data = {
	1536135941922: 'true',
	1536135962942: 'false',
	1536135986966: 'false',
	1536135989968: 'true'
}

let result = Object.entries(data).map(([k, v]) => ({ [k]: v }))

console.log(result)
```
