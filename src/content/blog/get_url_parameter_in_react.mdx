## get url parameter in react js

### Solution

```javascript
import React from 'react'
import { useLocation } from 'react-router-dom'

const MyComponent = () => {
	const search = useLocation().search
	const id = new URLSearchParams(search).get('id')
	console.log(id) //12345
}
```
