## nodejs fs write line

### Solution

```javascript
import fs from 'fs'

const fileWritter = fs.createWriteStream('./users-data.txt' {
  flags: 'a' // 'a' means appending (old data will be preserved)
})

// Will append one line to the file
fileWritter.write(JSON.stringify({
}))
```
