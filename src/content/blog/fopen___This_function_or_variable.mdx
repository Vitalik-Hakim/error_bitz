## `fopen`: This function or variable may be unsafe. Consider using fopen_s instead. To disable deprecation, use \_CRT_SECURE_NO_WARNINGS

### Solution

```cpp
#define _CRT_SECURE_NO_DEPRECATE
#include <stdio.h>
```
