## aws sdk s3 copy object bucket not found

### Solution

```whatever
You have to specify the bucket inside the copy source

// Copy an object.
$s3->copyObject(array(
    'Bucket'     => $targetBucket,
    'Key'        => $targetKeyname,
    'CopySource' => "{$sourceBucket}/{$sourceKeyname}",
));
```
