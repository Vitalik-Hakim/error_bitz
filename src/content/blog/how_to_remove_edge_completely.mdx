## how to remove edge completely from ubuntu

### Solution

```shell
sudo apt remove microsoft-edge-stable;
sudo apt purge microsoft-edge-stable;
cd ~/.config/ ;
rm -rf microsoft-edge/;
cd ~/.cache/;
rm -rf microsoft-edge;
rm -rf Microsoft/
```
