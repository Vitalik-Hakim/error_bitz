## cari kata pada string php

### Solution

```sql
$string = "Ini adalah contoh string untuk mencari kata.";
$keyword = "CONTOH";

$found = stristr($string, $keyword);

if ($found !== false) {
    echo "Kata ditemukan: $found";
} else {
    echo "Kata tidak ditemukan.";
}
```
