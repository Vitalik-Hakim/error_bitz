## update tooltip jquery

### Solution

```javascript
$('#topic_1').tooltip({ title: 'Hello' }).tooltip('show')
setTimeout(function () {
	$('#topic_1').tooltip('dispose').tooltip({ title: 'Goodbye' }).tooltip('show')
}, 5000)
```
