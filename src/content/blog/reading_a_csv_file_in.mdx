## reading a csv file in python

### Solution

```python
import csv

# open the file
with open('csvfile.csv' , 'r') as csvfile:
    # create the object of csv.reader()
    csv_file_reader = csv.reader(csvfile,delimiter=',')
    for row in csv_file_reader:
        print(row)
```
