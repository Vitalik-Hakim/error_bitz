## get array if value exists in multidimensional array

### Solution

```typescript
<?php
$user_array = array(
    0 => array(
        "name" => "kumar Sai",
        "username" => "kskumar",
        "id" => 1011
    ),
    1 => array(
        "name" => "badri",
        "username" => "badrikumar",
        "id" => 1012
    ),
    2 => array(
        "name" => "suresh",
        "username" => "skumar",
        "id" => 1013
    ),
    3 => array(
        "name" => "suresh",
        "username" => "dkumar",
        "id" => 1013
    )
);
$new_array = array();
if (array_search('suresh', array_column($user_array, 'name')) !== FALSE)
{
  $keys = array_keys(array_column($user_array, 'name'), 'suresh');
  $new_array = array_map(function($k) use ($user_array){return $user_array[$k];}, $keys);
}
else
    echo 'User Not Available';

print_r($new_array);
?>
```
