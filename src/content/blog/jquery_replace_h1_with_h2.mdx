**The solution for this is noted below**

## jquery replace h1 with h2

### Solution

```javascript
$(selector).load(url, function () {
	// your existing load function
	$(selector)
		.find('h1')
		.replaceWith(function () {
			return '<h2>' + $(this).text() + '</h2>'
		})
})
```
