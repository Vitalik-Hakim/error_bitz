## checkbox on click jquery

### Solution

```javascript
$(document).ready(function () {
	//set initial state.
	$('#textbox1').val($(this).is(':checked'))

	$('#checkbox1').change(function () {
		$('#textbox1').val($(this).is(':checked'))
	})

	$('#checkbox1').click(function () {
		if (!$(this).is(':checked')) {
			return confirm('Are you sure?')
		}
	})
})
```
