**The solution for this is noted below**

## Datatable Change page size for pdf

### Solution

```javascript
buttons: [
            'csv', 'excel',
            {
                extend: 'pdfHtml5',
                pageSize: 'A3',
                exportOptions: {
                    columns:':visible',
                    rows: ':visible'
                }
            },
        ],
```
**Try other methods by searching on the site. That is if this doesn't work**
