## mp4 to mp3 in python

### Solution

```python
# -*- coding: utf-8 -*-
from moviepy.editor import *

video = VideoFileClip('Sora no Kiseki the 3rd Evolution [BGM RIP] - Cry for your Eternity.mp4')
video.audio.write_audiofile('test.mp3')
```
