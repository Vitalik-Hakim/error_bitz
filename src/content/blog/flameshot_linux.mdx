## flameshot linux

### Solution

```shell
# Arch:
pacman --sync flameshot

# Debian 10+:
apt install flameshot

# Ubuntu 18.04+:
apt install flameshot

# Fedora:
dnf install flameshot

# NixOS:
nix-env --install --attr nixos.flameshot

# openSUSE:
zypper install flameshot

# Void Linux:
xbps-install flameshot

# Solus:
eopkg install flameshot
```
