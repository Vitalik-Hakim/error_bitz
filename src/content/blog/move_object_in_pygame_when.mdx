## move object in pygame when keydown and stop when keyup

### Solution

```python
x = 0
x_move = 0
while True:
	for event in pygame.event.get():
		if event.type == KEYDOWN:
    		if event.key == K_RIGHT:
        		x_move = 5
        elif event.type == KEYUP:
        	x_move = 0
     x += x_move
```
