## install pip ubuntu

### Solution

```shell
apt upgrade && apt update
apt install python3
python3 -m pip install -U pip   # OR sudo apt install python3-pip
python3 -m pip --version
pip --version
```

**Try other methods by searching on the site. That is if this doesn't work**
