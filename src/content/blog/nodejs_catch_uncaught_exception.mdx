## nodejs catch uncaught exception

### Solution

```javascript
process
	.on('uncaughtException', (err) => console.error(err, 'Uncaught Exception'))
	.on('unhandledRejection', (reason, promise) =>
		console.error(reason, 'Unhandled Rejection at', promise)
	)
```
