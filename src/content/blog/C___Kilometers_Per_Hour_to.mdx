## C++ Kilometers Per Hour to Miles Per Hour Conversion

### Solution

```cpp
const float A = 1.609344;
    float k, mph;

    cout << "Input the distance in Kilometer: ";
    cin >> k;

    mph = (k / A);

    cout << "The result is " << mph << " miles per hour" << endl;
```
