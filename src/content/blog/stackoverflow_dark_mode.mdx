## stackoverflow dark mode

### Solution

```whatever
Set stackoverflow to dark theme
1. Select user profile icon (in stackoverflow)
2. Select the 'Edit profile and settings' tab
3. Under 'SITE SETTINGS' menu, select 'Preferences'
4. Choose Theme: Light, System Setting or Dark.
```
**Try other methods by searching on the site. That is if this doesn't work**
