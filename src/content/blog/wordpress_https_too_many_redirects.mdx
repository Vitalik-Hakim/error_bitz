## wordpress https too many redirects

### Solution

```php
define('WP_HOME','https://mywebsite.com');
define('WP_SITEURL','https://mywebsite.com');
$_SERVER['HTTPS'] = 'on';
```

**Try other methods by searching on the site. That is if this doesn't work**
