## golang check file extension

### Solution

```go
import (
	"fmt"
	"path/filepath"
 )

func main(){
	path := "/media/godfather.mp4"

	fileExtension := filepath.Ext(path)

	fmt.Println("File extension ", fileExtension)

  }
```
