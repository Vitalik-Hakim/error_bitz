## unity main texture not working

### Solution

```csharp
//Get object renderer
rend = GetComponent<MeshRenderer>();

//deprecated do not use
rend.material.maintexture = newTexture;

//proper method
rend.material.SetTexture("_BaseMap",newTexture);
```
