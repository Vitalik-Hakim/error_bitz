**The solution for this is noted below**

## Error: Invalid src prop (https://i.ytimg.com/vi/ZKmJb_p0mHo/mqdefault.jpg) on `next/image`, hostname "i.ytimg.com" is not configured under images in your `next.config.js`

### Solution

```javascript
/** @type {import('next').NextConfig} */
module.exports = {
	reactStrictMode: true,
	images: {
		remotePatterns: [
			{
				protocol: 'https',
				hostname: '**'
			}
		]
	}
}
```

**Try other methods by searching on the site. That is if this doesn't work**
