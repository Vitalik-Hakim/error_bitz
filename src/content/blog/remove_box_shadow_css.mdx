**The solution for this is noted below**

## remove box shadow css

### Solution

```css
/* remove shadows by xomisse */
.selector{
	-moz-box-shadow: none;
	-goog-ms-box-shadow: none;
	-webkit-box-shadow: none;
	box-shadow: none;
}
```
