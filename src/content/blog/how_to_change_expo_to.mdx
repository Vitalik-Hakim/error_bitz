## how to change expo to tunnel

### Solution

```whatever
// use:
expo start --tunnel
// you can make it by default by going into package.json in your project and
// by changing lines:
"scripts": {
    "start": "expo start",
// to:
"scripts": {
    "start": "expo start --tunnel",
// and from now on in terminal instead of "expo start --tunnel" you can use:
npm start
```
