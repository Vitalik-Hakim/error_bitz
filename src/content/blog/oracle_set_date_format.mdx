## oracle set date format

### Solution

```sql
ALTER SESSION SET NLS_DATE_FORMAT = 'DD-MM-YYYY HH24:MI:SS';
SELECT sysdate FROM dual;

SELECT to_char(sysdate,'HH24:MI:SS') FROM dual;
```
