## javascript stack last element

### Solution

```javascript
let stack = []

stack.push(1)
stack.push(2)
stack.push(3)
stack.push(4)

let i = stack.pop()
let lastElement = stack[stack.length - 1]

console.log(i, lastElement) // i = 4, lastElement = 3
```
