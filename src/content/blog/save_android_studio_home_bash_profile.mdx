## save android studio home bash_profile

### Solution

```shell
nano ~/.bash_profile

export ANDROID_HOME=/YOUR_PATH_TO/android-sdk
export PATH=$ANDROID_HOME/platform-tools:$PATH
export PATH=$ANDROID_HOME/tools:$PATH

source ~/.bash_profile
echo $ANDROID_HOME
```
