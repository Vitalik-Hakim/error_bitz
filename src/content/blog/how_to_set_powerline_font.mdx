## how to set powerline font in vscode

### Solution

```whatever
/* Download a powerline font like Cascadia Code PL and then add it to your 
                        VsCode settings.json using                         */
   
{
"terminal.integrated.fontFamily": "Cascadia Code PL",
}
```
