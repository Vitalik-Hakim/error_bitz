## Fetch Await

### Solution

```javascript
async function handleFetchAwait(id) {
	let response = await fetch(`https://samplesite.com/users_id/${id}`)
	let data = await response.json()
	return data
}

handleFetchAwait(21).then((data) => console.log(data))
```
