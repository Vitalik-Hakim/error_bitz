## css for internet explorer only

### Solution

```css
@media all and (-ms-high-contrast: none), (-ms-high-contrast: active) {
	/* IE10+ CSS styles goes underneath */
	#ie11 h2 {
		color: #fff;
	}
	#ie11 h2:hover {
		color: gray;
	}
}
```
