## uptime kuma docker compose

### Solution

```shell
version: "3.9"
services:
  app:
    container_name: uptime-kuma
    image: louislam/uptime-kuma
    ports:
      - "3001:3001"
    volumes:
      - /opt/uptime-kuma:/app/data
      #- /portainer/Files/AppData/uptime-kuma:/app/data
    restart: unless-stopped
```
