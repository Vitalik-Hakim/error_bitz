## gatsby js install

### Solution

```shell
npm install -g gatsby-cli


//Create new project
gatsby new [gatsby-project-name] https://github.com/gatsbyjs/gatsby-starter-hello-world
cd [gatsby-project-name]

//initialize developer server
gatsby develop

//create production build the project
gatsby build
gatsby serve
```
